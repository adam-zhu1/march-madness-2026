# 🏀 March Madness 2026 Bracket Predictor

> A data-driven NCAA men’s tournament prediction app built to help make smarter picks for the **2026 ESPN Tournament Challenge**.

---

## ✨ Overview

This project combines:

- historical NCAA tournament matchup results
- advanced team ratings
- a trained logistic regression model
- a Streamlit app for interactive bracket decision-making

The result is a local app that helps you:

- compare any two tournament teams
- view first-round recommendations
- inspect power rankings
- make one-by-one bracket decisions more confidently

---

## 🚀 What This Project Does

### 📊 Power Rankings
Ranks tournament teams using merged team-level metrics.

### 🧠 Matchup Prediction
Predicts the winner between any two teams using a trained historical model.

### 📝 Round of 64 Recommendations
Provides recommended first-round picks with estimated win probabilities.

### 🖥️ Local App Interface
Displays everything in a Streamlit app so you can use it while filling out your bracket.
